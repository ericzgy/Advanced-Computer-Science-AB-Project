    Cougar Space

    Cougar Space is a collaborative learning project by Eric Guan and Saim Majid that seeks to provide the same functionality that applications such as momentum provide, but for the Oakton High School student population.

    The current website is a work in progress and is in the alpha testing stage, however at the moment it has a suite of features such as a to-do list, weather display, OHS news, and a calendar with tasts. The entire site was built using HTML/CSS/Javascript. As development continues, expect more features to be added! Find the project here.
